# 🙏 해운사 절 전용 불사 조회 앱

절에서 불사내용을 조회할 수 있는 전용 웹 앱입니다.

## 🌟 특징

- ✅ 절 체크된 불사내용만 표시
- 🔒 금액 정보 숨김 (조회 전용)
- 📱 모바일 최적화
- 🔄 실시간 데이터 동기화
- 🎨 직관적인 UI

## 🚀 배포 방법

### Vercel로 배포 (추천)

1. GitHub에 코드 업로드
2. https://vercel.com 에서 Import
3. 자동 배포 완료!

자세한 내용은 `DEPLOYMENT_GUIDE.md` 참고

## 🔐 로그인 정보

```bash

```
